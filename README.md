# E-commerce Conversion Intelligence Platform

End-to-end e-commerce funnel analytics platform with GenAI-powered insights.

## 🎯 Project Overview
Analyzing clickstream events to identify conversion bottlenecks and revenue opportunities using SQL, dbt, Python, GenAI, and Power BI.

## 🛠 Tech Stack
- **Languages:** Python, SQL
- **Database:** PostgreSQL (Supabase)
- **Transformation:** dbt
- **Analytics:** Pandas, NumPy
- **GenAI:** Google Gemini API
- **Visualization:** Power BI, Streamlit
- **Deployment:** Streamlit Cloud
